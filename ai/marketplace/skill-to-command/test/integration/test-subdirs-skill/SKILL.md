# Test Subdirectories Skill

This skill tests the DIRECTORY_WITH_SUBDIRS transformation mode.

## Features

- Contains subdirectories that should be preserved
- Multiple markdown files that should be merged
- Scripts in subdirectories (not root level)

## Usage

This skill demonstrates how subdirectories like `scripts/` and `lib/` are preserved.
